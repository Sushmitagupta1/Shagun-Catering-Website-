# Shagun Catering — Serving Joy Since 1989

Ahmedabad's premier catering service, a division of **Upper Crust**. Specializing in weddings, corporate events, sangeet, and intimate gatherings — from 500 to 5,000+ guests.

## Features

- Responsive catering website with elegant UI
- Cuisine showcase (Indian, Italian, Chinese, Mediterranean, Mexican, Thai)
- Event service cards (Wedding, Corporate, Sangeet, Birthday, Anniversary, House Party)
- Downloadable packages PDF
- Photo gallery with filter
- Contact form & WhatsApp integration
- Floating CTAs for mobile

## Pages

| Page | Description |
|------|-------------|
| `index.html` | Home — hero, cuisines, services, process, testimonials |
| `about.html` | Story, founders, legacy |
| `packages.html` | Catering packages & menu details |
| `gallery.html` | Event photo gallery |
| `contact.html` | Contact form, info, map |

## Tech Stack

- HTML5, CSS3, vanilla JavaScript
- Font Awesome 6, Google Fonts (Playfair Display, Inter)
- Hosted via static file server or any web host
